**Public Access Catalog System**
A simple Public Access Catalog System that allows users to search for books and view their availability.

**Features**

User
- Search for books
- View book availability

Admin
- Add new books
- Update book information
- Delete books
- Basic CRUD operations for catalog management

**Tech Stack**
- Java
- SQL
- XAMPP (MySQL)

**Notes**

This project was built as a learning exercise to practice database integration and basic CRUD operations using Java.
